MANUAL GITHUB UPLOAD COMMANDS
================================

If the push.bat file doesn't work, copy and paste these commands one by one:

1. Initialize Git:
git init

2. Add all files:
git add .

3. Create commit:
git commit -m "Initial commit: AI-Powered Web Firewall by Akash Hossain (arknox.in)"

4. Set main branch:
git branch -M main

5. Add remote repository:
git remote add origin https://github.com/mdakashhossain1/AI-Powered-Web-Firewall.git

6. Push to GitHub:
git push -u origin main

================================

ALTERNATIVE: Use GitHub Desktop
1. Download: https://desktop.github.com/
2. Clone: https://github.com/mdakashhossain1/AI-Powered-Web-Firewall.git
3. Copy your files to the cloned folder
4. Commit and push through the GUI
