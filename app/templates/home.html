About Our Project

This project was developed to created an AI-powered firewall that uses machine learning algorithms to identify and block potentially malicious web traffic.

Project Overview

Our AI Firewall analyzes incoming web requests using a trained machine learning model to distinguish between legitimate users and potential threats. The system evaluates multiple features from each request and makes real-time decisions to protect web applications from various attacks.

The project demonstrates the practical application of machine learning in cybersecurity and showcases how AI can enhance traditional security measures.

Our Approach

We trained our model using the NSL-KDD dataset, which contains thousands of network traffic samples labeled as either normal or attack traffic. Our Random Forest classifier achieved high accuracy in identifying malicious requests.

The web application is built with Flask and features a user-friendly interface that provides real-time feedback. We've implemented a verification system that helps distinguish between human users and automated attacks.

Technical Implementation

Our project uses a Random Forest classifier trained on the NSL-KDD dataset to identify potential threats. The model analyzes 41 different features extracted from each web request to make its prediction.

The web application is built with Flask, a lightweight Python web framework, and uses Bootstrap for responsive design. We've implemented client-side verification using JavaScript to enhance the user experience and provide an additional layer of security.

The entire codebase follows best practices for security and performance, with modular architecture that allows for easy maintenance and future enhancements.
